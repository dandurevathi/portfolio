Abstract

This project presents a desktop application developed in C using the Windows API that predicts the genre of Telugu songs based on their musical features and enables instant playback via YouTube. The system reads a dataset containing song attributes such as danceability, acousticness, energy, and popularity. These inputs are processed through a rule-based genre prediction model, neural networks with initialized weights. The graphical user interface allows users to enter a song name, view its predicted genre, and play the song on YouTube with a single click. This project uniquely combines machine learning logic with user interactivity, offering both educational and entertainment value. It demonstrates real-time genre classification and multimedia integration, showcasing a practical use of data science and C programming in multimedia applications.
